About me

My name is Erik Schaal. But my Github user name is Techno-mage. I am a full stack developer, developing in the node enviornment. I graduated from the University of Minnesota with a B.A. in computer science, and I have recently completed a 3 month web developer bootcamp learning the MERN stack. This bootcamp was held remotely, giving me experinece as a remote developer using zoom, and the Live Share extention for VS code.

My front end tech stack includes HTML5, CSS, bootstrap, javascript, jquery, react, and the use of third party API's. My back end technologies include the node.js, express servers, mySQL database using the sequelize ORM, and mongoDB using the Mongoose ORM.

When I am not programing, my hobbies include video games, table top roleplaying games, anime, and homebrewing beer.
